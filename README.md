# Operations Quality Analytics Dashboard

## Project Overview

This project demonstrates an end-to-end Business Intelligence solution built using **PostgreSQL, SQL, Power BI, and Excel** to analyze operational audit performance across multiple partners and business functions.

The objective was to transform raw audit data into meaningful business insights through data modeling, KPI development, SQL-based analysis, and interactive dashboard visualization.

---

## Project Objectives

The project was designed to answer key operational questions:

* Which partners handle the highest audit volume?
* Which partners consistently achieve better quality scores?
* How do fatal error rates vary across operations?
* How does performance differ between business functions?
* What trends can be observed over time?
* Which areas require operational improvement?

## Dashboard Pages

### Executive Overview

The Executive Overview dashboard provides a high-level summary of operational performance.

Features include:

* Total Audit Volume
* Average Audit Score
* Fatal Percentage
* Partner Performance Comparison
* Monthly Quality Trends
* Key Business Insights

### Business Function Analysis

The second dashboard page focuses on comparative performance analysis between business functions.

Features include:

* Audit Volume Comparison
* Quality Score Comparison
* Fatal Percentage Analysis
* Monthly Trend Analysis
* Partner Performance Across Functions

---

## Key Insights

* Partner Alpha handled the highest audit volume.
* Partner Beta achieved the highest average audit score.
* Partner Epsilon recorded the lowest fatal percentage.
* Customer Operations consistently outperformed Merchant Operations in quality metrics.
* Audit quality showed significant improvement during the final reporting period.
