Project Name

Fintech Audit Analytics Dashboard

Overview

A Business Intelligence project built using PostgreSQL and Power BI to analyze audit quality, partner performance, fatal error rates, and operational trends across multiple Lines of Business.

Tools Used

• PostgreSQL
• SQL
• Power BI
• Excel

Dataset

• 11,811 audit records
• 5 partners
• 2 LOBs (CX & MHD)
• Reporting Period: Jan 2025 - Dec 2025

Key Insights

• TRANS handled the highest audit volume.
• Concentrix achieved the highest audit score.
• Startek recorded the lowest fatal rate.
• CX outperformed MHD in audit quality.
